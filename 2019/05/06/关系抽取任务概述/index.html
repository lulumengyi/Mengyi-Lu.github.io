关系抽取任务概述

这也是一个关系抽取的任务。
什么是关系抽取呢？主要负责从文本中识别出实体（Entities)，抽取实体之间的语义关系。例如，句子“Bill Gates is the founder of Microsoft Inc.”中包含一个实体对(Bill Gates, Microsoft Inc.)，这两个实体对之间的关系为Founder。

在本文中如：”The news brought about a commotion in the office.这段话标明两个实体（news，commotion）这两个实体之间的关系是Cause-Effect(e1,e2)。因果果关系。
1.SemEval-2010 Task 8: Multi-Way Classiﬁcation of Semantic Relations Between Pairs of Nominals 多对名词之间语义关系的分类
有9种关系如下所示:

Cause-Effect (CE).An event or object leads to an effect. Example: those cancers were caused by radiation exposures.因果关系

Instrument-Agency (IA). An agent uses an instrument. Example: phone operator

Product-Producer (PP).A producer causes a product to exist. Example: a factory manufactures suits 生产与被生产的关系

Content-Container (CC).An object is physically stored in a delineated area of space. Example:a bottle full of honey was weighed 容器与内容物的关系

Entity-Origin (EO).An entity is coming or is derived from an origin (e.g., position or mate-rial) Example:letters from foreign countries 实体来自或源自原产地（例如，位置或材料）

Entity-Destination (ED).An entity is moving towards a destination. Example: the boy went to bed 一个实体正在向目的地移动。 例如：男孩去睡觉

Component-Whole (CW).An object is a component of a larger whole. Example: my apartment has a large kitchen 组件到整体

Member-Collection (MC).A member forms anonfunctional part of a collection. Example:there are many trees in the forest 成员集合关系（MC），成员构成集合的非功能部分。 例如：森林里有很多树

Message-Topic (MT).A message, written or spoken, is about a topic. Example: the lecture was about semantics 。信息与主题 例如：讲座是关于语义的。

Bidirectional Network

一个lstm结构如上图所示。首先我们来理解下什么是lstm，在讲lstm之前，我们先讲下rnn。全连接神经网络和卷积神经网络他们都只能单独的取处理一个个的输入，前一个输入和后一个输入是完全没有关系的。但是，某些任务需要能够更好的处理序列的信息，即前面的输入和后面的输入是有关系的。比如我们对一个句子的理解就需要知道前后的单词的意思。RNN 可以被看做是同一神经网络的多次赋值,每个神经网络模块会把消息传递给下一个，

如果把上面的那个带箭头的圈去掉，它就变成了最普通的全连接神经网络。x是一个向量，它表示输入层的值（这里面没有画出来表示神经元节点的圆圈）；s是一个向量，它表示隐藏层的值（这里隐藏层面画了一个节点，你也可以想象这一层其实是多个节点，节点数与向量s的维度相同）；U是输入层到隐藏层的权重矩阵；o也是一个向量，它表示输出层的值；V是隐藏层到输出层的权重矩阵。那么，现在我们来看看W是什么。循环神经网络的隐藏层的值s不仅仅取决于当前这次的输入x，还取决于上一次隐藏层的值s。权重矩阵 W就是隐藏层上一次的值作为这一次的输入的权重。

按时间序列展开后大概是这样的结构，这样看就比较清楚了，输入是xt，隐藏值是st，输出值是ot，st的值不仅取决于xt，还取决于st-1。用公式表示循环神经网络的话：
ot=g(Vst) –[1]
st=f(Uxt+Wst-1) –[2]
式1是输出层的计算公式，输出层是全连接层，也就是每个结点都和隐藏层节点相连，g是激活函数。V是输出层的权重矩阵。式2是隐藏层的计算公式，也是循环层，U是输入x的权重矩阵，W是st-1作为输入的权重矩阵，f是激活函数。

我们也可以看出，循环层和全连接层多了一个权重矩阵W。
但RNN有个弊端就是无法解决长距离依赖问题，并不能很好的处理较长的序列。一个主要的原因是，RNN在训练中很容易发生梯度爆炸和梯度消失，这导致训练时梯度不能在较长序列中一直传递下去，从而使RNN无法捕捉到长距离的影响。为什么RNN会产生梯度爆炸和消失问题呢？公式可以推导出来，权重数组W最终的梯度是各个时刻的梯度之和，在某个时刻的时候梯度就下降为0了，那么，从这个时刻开始再往之前走，得到的梯度（几乎为零）就不会对最终的梯度值有任何贡献。

详细

lstm就可以解决长距离依赖这个问题。原始RNN的隐藏层只有一个状态，即h，它对于短期的输入非常敏感。那么，假如我们再增加一个状态，即c，让它来保存长期的状态，新增加的状态c，称为单元状态(cell state)。
LSTM的关键，就是怎样控制长期状态c。在这里，LSTM的思路是使用三个控制开关。第一个开关，负责控制继续保存长期状态c；第二个开关，负责控制把即时状态输入到长期状态c；第三个开关，负责控制是否把长期状态c作为当前的LSTM的输出。开关是怎样在算法中实现的呢？这就用到了门（gate）的概念。门实际上就是一层全连接层，它的输入是一个向量，输出是一个0到1之间的实数向量。假设W是门的权重向量，是偏置项，那么门可以表示为：

本文中总体框架如下：

输入层：将原始句子输入该层；

向量层：将每个单词映射到一个低维向量；

LSTM层：利用BLSTM从输入的向量得到该句子的强特征；

att层：产生一个权重向量，将LSTM中的每一个时间节点通过这个权重向量联结起来；

输出层：将上面得到的向量运用到关系分类任务上。

Attention

什么叫attention？跟我们人类注意力机制差不多，比如我们在过马路时，虽然我们视野会很大，但我们会比较注意红绿灯的状态。
在深度学习中我们的注意力机制怎么实现的呢，比如我们要生成目标句子某个单词，就是说“汤姆”对应的概率分布：计算生成Yi时的输入句子单词“Tom”、“Chase”、“Jerry”对Yi来说的注意力分配概率分布。
attention矩阵可以通过一个线性层+softmax得到，也就是红框中的部分，其输入是decoder上一时刻的状态si-1以及encoder对应当前第j个词的输出hj。其意思就是当前词的权重，由当前词和decoder的所翻译的前一个词决定，同时引入了输入和输出的信息。
